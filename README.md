## Getting Started

### 1. Clone the repo

```bash
git https://github.com/chosenvincent1/Middleware-Nodejs-APM.git
cd Middleware-Nodejs-APM
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Add Middleware Credentials

serviceName= your-service-name
accessToken=your_middleware_api_key

### 4. Start server

```bash
node index.js
```
